## 项目管理

### 创建项目
- 填写项目名称（必填）和描述（可选）
- 创建后自动生成项目ID和时间戳

### 编辑项目
- 点击项目卡片上的编辑按钮进入编辑模式
- 修改项目名称和描述
- 编辑模式下表单背景变为黄色以示区分
- 支持取消编辑返回原状态

### 项目详情
每个项目提供四个管理入口：

| 入口 | 功能 |
|------|------|
| 时间线 | 查看项目相关的所有事件记录 |
| 活动 | 管理项目活动和任务进度 |
| 文件 | 项目文件存储和版本管理 |
| 联系人 | 管理与项目关联的联系人 |

### 项目联系人
- 支持手动添加联系人到项目
- 可设置联系人在项目中的角色和备注
- **自动绑定机制**：创建事件时关联的联系人会自动绑定到对应项目

## 联系人管理

### 基本信息
| 字段 | 说明 | 必填 |
|------|------|------|
| 姓名 | 联系人姓名 | 是 |
| 职位 | 工作职位 | 否 |
| 单位/公司 | 所属组织 | 否 |
| 邮箱 | 电子邮件地址 | 否 |
| 电话 | 支持添加多个电话号码 | 否 |
| 地址 | 联系地址 | 否 |

### 扩展信息
- **标签**：用逗号分隔多个标签，便于分类管理
- **备注**：记录背景信息、擅长领域、沟通风格等

### 编辑联系人
- 点击联系人卡片上的编辑按钮进入编辑模式
- 可修改所有联系人信息
- 编辑模式下表单背景变为黄色
- 支持取消编辑返回原状态

### 联系人时间线
- 点击"查看时间线"查看与该联系人相关的所有事件
- 按时间倒序展示互动历史
- 显示其他参与者信息

## 事件记录

### 创建事件
| 字段 | 说明 | 必填 |
|------|------|------|
| 事件标题 | 简要描述事件内容 | 是 |
| 日期 | 事件发生的日期 | 是 |
| 类型 | 选择事件分类 | 否 |
| 关联项目 | 可选择关联到某个项目 | 否 |
| 相关联系人 | 参与事件的联系人 | 是（至少1人） |
| 描述 | 详细记录事件内容 | 否 |
| 提醒时间 | 设置提醒通知时间 | 否 |

### 编辑事件
- 点击事件卡片上的编辑按钮进入编辑模式
- 可修改事件的所有信息，包括关联联系人
- 编辑模式下表单背景变为黄色
- 支持取消编辑返回原状态

### 事件类型
支持以下13种事件类型：

| 沟通类 | 协作类 | 其他 |
|--------|--------|------|
| 会议 | 里程碑 | 个人 |
| 电话 | 工单 | 其他 |
| 邮件 | 视频会议 | |
| 微信 | | |
| QQ | | |
| 钉钉 | | |
| 飞书 | | |
| 短信 | | |

### 联系人快速选择
- 支持模糊搜索快速定位联系人
- 自动补全建议，提升选择效率
- 适合联系人数量较多的场景

### 自动绑定
当事件关联了项目时，事件中的所有联系人会**自动绑定到该项目**，无需手动在项目中再次添加。

## 事件提醒

### 设置提醒
1. 在创建或编辑事件时，勾选"设置提醒"
2. 选择提醒的日期和时间
3. 到达设定时间时会发送系统通知

### 今日提醒
- 设置了今日提醒的事件会在事件列表顶部**置顶显示**
- 以橙色高亮区分，带有钟表图标
- 显示具体的提醒时间

### 提醒通知
- 使用系统原生通知
- 支持后台提醒（应用最小化时也能收到通知）

## 工作总结

### 总结类型
支持以下5种总结类型：

| 类型 | 说明 |
|------|------|
| 日总结 | 单日工作总结 |
| 周总结 | 一周工作总结 |
| 月总结 | 单月工作总结 |
| 年总结 | 年度工作总结 |
| 自定义 | 自定义时间范围 |

### 生成总结
1. 在"总结"标签页选择总结类型
2. 选择日期范围（日/周/月/年会自动计算范围）
3. 点击"生成总结"
4. 系统自动汇总该时间段内的事件记录

### 总结内容
生成的总结包含：
- 事件数量统计
- 按日期分组的事件列表
- 参与的联系人信息
- 关联的项目信息

### 历史总结
- 左侧显示历史总结列表
- 点击可查看总结详情
- 支持删除不需要的总结
- 自动生成的总结标记为"自动"

## 活动管理

### 活动状态流转
```
待分配 → 未激活 → 进行中 → 已完成
                    ↓
                  已暂停 → 进行中
```

| 状态 | 说明 | 可执行操作 |
|------|------|------------|
| 待分配 | 未指定负责人 | 分配负责人 |
| 未激活 | 已分配负责人但未开始 | 激活 |
| 进行中 | 已激活正在执行 | 暂停、完成 |
| 已暂停 | 暂时中止 | 重新激活 |
| 已完成 | 任务已完成 | - |

### 创建活动
| 字段 | 说明 | 必填 |
|------|------|------|
| 活动名称 | 活动标题 | 是 |
| 活动描述 | 详细说明 | 否 |
| 预计完成时间 | 目标完成日期 | 否 |
| 负责人 | 可多选（仅限项目联系人） | 否 |

### 负责人限制
活动的负责人**必须是已绑定到项目的联系人**，确保团队成员与项目的关联性。

### 甘特图导出
- 支持导出项目活动为 Excel 格式的甘特图
- 包含活动信息和时间线标记
- 图例说明：
  - ■ 待分配/未激活
  - ● 进行中
  - ◆ 已暂停
  - ★ 已完成

## 文件管理

### 文件上传
- 支持拖拽上传或点击选择文件
- 文件自动复制到项目专属目录
- 目录命名格式：`项目名称_项目ID`

### 版本管理
- 同名文件自动创建新版本
- 保留历史版本，带时间戳标记
- 版本号自动递增，显示为 `v2`、`v3` 等

### 文件操作
| 操作 | 说明 |
|------|------|
| 打开 | 使用系统默认程序打开文件 |
| 定位 | 在文件管理器中显示文件位置 |
| 删除 | 删除文件（不可恢复） |

### 全局文件搜索
- 在事件标签页左侧可进行全局文件搜索
- 支持跨项目搜索文件
- 按文件名模糊匹配
- 显示文件所属项目

## 搜索功能

### 列表搜索
各列表页面均支持搜索功能：

| 列表类型 | 搜索字段 |
|----------|----------|
| 项目列表 | 项目名称、描述 |
| 事件列表 | 事件标题、描述、类型、关联项目 |
| 联系人列表 | 姓名、职位、公司、标签 |

### 搜索交互
1. 输入关键词后显示匹配建议（下拉列表）
2. 支持键盘导航（↑↓ 选择，Enter 确认，Esc 关闭）
3. 选中后自动定位到对应项并高亮显示（3秒后消失）

### 事件表单联系人搜索
- 在创建/编辑事件时支持搜索联系人
- 模糊匹配姓名和职位
- 选中后自动添加到已选列表

## 多语言支持

### 支持语言
| 语言 | 标识 |
|------|------|
| 简体中文 | 中文 |
| English | EN |

### 切换方式
1. 在页面右上角找到语言切换按钮
2. 点击对应语言切换
3. 界面立即更新为所选语言
4. 设置自动保存到本地存储

- 导航标签
- 表单标签和提示
- 按钮文本
- 提示消息
- 弹窗内容
- 空状态提示

## 数据安全

### 本地存储
- 所有数据存储在用户设备本地
- 数据库位置：
  - macOS: `~/Library/Application Support/mindmirror/`
  - Windows: `%APPDATA%\mindmirror\`
  - Linux: `~/.local/share/mindmirror/`
- 文件存储：`{数据目录}/project_files/`

### 隐私保护
- 不收集任何用户数据
- 不上传数据到服务器
- 完全离线可用
- 无需网络连接

### 数据格式
| 数据类型 | 格式 |
|----------|------|
| 数据库 | SQLite |
| 文件 | 原始格式保存 |
| 活动导出 | Excel (.xlsx) |
| 数据导出 | JSON |

### 数据备份建议
定期备份以下内容：
1. 数据库文件：`mindmirror_local.db`
2. 项目文件目录：`project_files/`
